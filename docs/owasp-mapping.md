# Cartographie OWASP pour la gouvernance des agents

Ce projet couvre les principales catégories de risque liées à l’IA agentique et à la sécurité Zero Trust :

- Agent-01: Prompt Injection
- Agent-03: Sensitive Data Exposure / Exfiltration
- Agent-06: Privilege Escalation

La politique de production et la politique de sécurité sont conçues pour bloquer ou restreindre ces comportements par défaut.
